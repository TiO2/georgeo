### Clone and install

First, clone the repo:

```
git clone https://github.com/rmpato/georgeo.git
```

Then, install it's dependencies.

```
npm install
```

Finally, run it.

```
npm start
```

Then you can browse  [localhost:3000](http://localhost:3000) and start geocoding.
